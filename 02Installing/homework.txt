 directory archive (tarball or "no-install"), or compiled methods?
	To install system-wide,using XAMPP integrated package, we will need to log in as a user with Windows administrator privileges.

3) How do you verify the integrity of downloaded packages?

	Once we have started the server, we is to set a password for the database root account.
